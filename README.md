# Simple ts with nodemon with Vitest

## Run with

- npm run dev
- npm run test:watch

## Added js packages for missing js features:

- @datastructures-js/priority-queue [https://github.com/datastructures-js/priority-queue/blob/v5/README.md]
    A performant priority queue implementation using a Heap data structure.
    - PriorityQueue that accepts a custom comparator between elements.
    - MinPriorityQueue which considers an element with smaller priority number as higher in priority.
    - MaxPriorityQueue which considers an element with bigger priority number as higher in priority.

- @datastructures-js/queue [https://github.com/datastructures-js/queue/tree/v4.2.3]
    A performant queue implementation in javascript.

- @datastructures-js/deque [https://github.com/datastructures-js/deque/tree/v1.0.4]
    A performant double-ended queue (deque) implementation in javascript.
